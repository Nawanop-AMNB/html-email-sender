# HTML Email Sender

#### Requirements:

node: ^18.17.0 \
package-manager: pnpm

##### Steps

1. put html file to ./test_html folder

2. add .env file (you can lookup for the .env.sample)

3. install dependencies and run start
    ```
    pnpm install
    ```
4. run start, select file and sent!
    ```
    pnpm start
    ```
